# Fisch Upper

Statyczna aplikacja webowa dla wedkarzy w Polsce:

- prognoza godzinowa dla dowolnych polskich wspolrzednych,
- obserwacje z publicznego API IMGW,
- najblizszy wodowskaz IMGW z poziomem wody, przeplywem i temperatura wody, jesli jest dostepna,
- interaktywna mapa lowisk,
- lokalny album zdjec,
- eksport i import zapisu JSON.

Otworz `index.html` w przegladarce. Aplikacja nie wymaga backendu ani procesu build.

Zrodla danych:

- Prognoza: https://open-meteo.com/en/docs
- Obserwacje IMGW: https://danepubliczne.imgw.pl/api/data/synop
- Wodowskazy IMGW: https://danepubliczne.imgw.pl/api/data/hydro
- OpenStreetMap: https://www.openstreetmap.org/
